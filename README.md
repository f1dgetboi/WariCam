## WariCam
使用したライブラリ：
#ultralytics 
皿などのオブジェクトデテクションのために使用
pip:
```
pip install ultralytics
```
#pygame
UIのために使用
pip:
```
pip install pygame
```
#Opencv
オブジェクトデテクションの可視化のために使用
pip:
```
pip install opencv-python
```
#mediapipe
手のトラッキングのために使用
pip:
```
pip install mediapipe
```
#pythonのバージョンについて
Python 3.9.13 64-bitを使用
